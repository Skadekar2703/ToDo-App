# 📱 Smart To-Do Android App

A modern and feature-rich To-Do application built using Kotlin that helps users manage daily tasks, track productivity, and stay consistent with their goals.

---

## ✨ Features

- 📝 Add daily tasks
- ✏️ Edit and delete tasks
- ✅ Mark tasks as completed
- 🧩 Subtasks support (shopping lists, study topics, etc.)
- 📊 Daily progress tracker
- 📅 Monthly completion tracker
- 🎨 Clean and minimal UI

---

## 📱 Screens

- Home Task List Screen
- Add Task Dialog
- Subtask Screen
- Monthly Calendar Tracker

---

## 🛠 Tech Stack

- Kotlin
- XML
- Room Database
- RecyclerView
- Material Design
